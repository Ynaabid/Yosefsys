Coloca aquí tus archivos de sonido para el portfolio.

Recomendaciones:
- El archivo click.mp3 debe estar en esta carpeta para el sonido de entrada (al hacer clic en recursos).
- El archivo click_out.mp3 debe estar en esta carpeta para el sonido de salida (al salir o cerrar recursos).
- Puedes descargar sonidos profesionales y libres de derechos desde sitios como https://mixkit.co/free-sound-effects/click/ o https://freesound.org/.
- Nómbralos exactamente como click.mp3 y click_out.mp3 para que el script los reconozca automáticamente.

Ejemplo de estructura:
assets/sounds/click.mp3
assets/sounds/click_out.mp3

Puedes añadir más sonidos y enlazarlos desde tu JS si lo deseas.
